# Inceptum — Systemic Survival 2D

![Alpha Badge](https://img.shields.io/badge/Status-Alpha_0.1.2-blueviolet)
![Tech Stack](https://img.shields.io/badge/Tech-React_|_TypeScript_|_Unity-blue)
![Discord](https://img.shields.io/badge/Community-Discord-7289da)

**Inceptum** (formerly Survivor IO) is a deep, systemic 2D survival game focused on realism, industrial progression, and ecosystem interaction. Inspired by classics like ARK, 7 Days to Die, and the tactical simplicity of MooMoo.io, it challenges players to survive a world that reacts to their every action.

## 🌟 Key Features

- **Systemic Crafting**: Over 55+ items with complex recipe chains ranging from primitive stone tools to advanced industrial machinery.
- **Dynamic Ecosystem**: Animals with realistic AI that react to day/night cycles, player presence, and physical barriers.
- **Deep Agriculture**: A multi-stage farming system including wheat, grapes, fermentation, and a complex nutritional chain.
- **Modular Construction**: Build persistent bases with modular blocks, defending them against environmental decay and future threats.
- **4-Player Co-op**: Optimized multiplayer experience via Unity Relay, featuring persistent worlds and synced seeds.

## 🛠 Tech Stack

- **Game Engine**: Unity (C#)
- **Web Frontend**: React + TypeScript (Vite)
- **Styling**: Vanilla CSS with modern Glassmorphism aesthetics
- **Animations**: Framer Motion
- **Internationalization**: i18next (Full PT/EN Support)
